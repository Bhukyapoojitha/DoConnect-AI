\# DoConnect AI



An intelligent discussion and collaboration platform inspired by StackOverflow and Discord.

----------------------

\## Features

----------------------

\- JWT Authentication

\- Role-Based Access Control

\- AI-Powered Question Answering

\- Real-Time Chat (WebSocket/STOMP)

\- Content Moderation

\- Admin Analytics Dashboard

\- Moderator Dashboard

\- Tag Management

----------------------------------

\## Tech Stack

-----------------------------------

\### Frontend

\- React

\- Redux Toolkit

\- Bootstrap

\- Axios



\### Backend

\- Spring Boot

\- Spring Security

\- JPA/Hibernate

\- MySQL

\- WebSocket
